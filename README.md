This repository is created to help beginners learn how to contribute to open source projects.

If you are new to GitHub, this repo will help you practice:
- Forking a repository
- Creating a branch
- Fixing small issues
- Making a pull request

## Who is this for?
- Beginners learning GitHub
- Students practicing open source workflow
- Anyone looking for their first contribution

## How to contribute
1. Fork this repository
2. Clone your fork
3. Create a new branch
4. Fix an issue
5. Submit a pull request

Please read `CONTRIBUTING.md` before submitting a PR.
